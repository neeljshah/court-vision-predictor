CourtVision API

Public REST API for multi-book NBA prop odds + model-graded EV picks. Free read-only. Rate limited 60 req/min/IP on HTML routes.

Odds endpoints

GET /api/odds                          # today's props, all books
GET /api/odds?stat=blk                 # filter by stat
GET /api/odds?player=wemby             # substring match player
GET /api/odds/{date}.json              # any date
GET /api/odds/{date}.csv               # CSV export for spreadsheets
GET /api/odds/best/{date}.json         # best book per side
GET /api/odds/summary/{date}           # compact snapshot (counts + freshness)
GET /api/odds/games/{date}             # distinct games in scrape
GET /api/odds/freshness/{date}         # per-book CSV age + row count
GET /api/odds/history/{player}/{stat}  # every captured quote, sorted by time
GET /api/odds/spread/{date}.json       # cross-book disagreement / arb finder
GET /api/odds/moves/{date}.json        # lines that moved in last window_minutes
GET /arbs                              # HTML view of /api/odds/spread

Model + picks endpoints

GET /api/slate?date={date}          # top 15 graded bets
GET /api/bet/{bet_id}               # single bet detail
GET /api/parlays?date={date}        # MC-priced parlays
GET /api/auto_parlay?stake=20       # best parlay for $stake
GET /api/plus_ev?min_ev_pct=5       # every (bet, book) over EV gate
GET /api/today_summary?n=3          # compact top-N for embeds
